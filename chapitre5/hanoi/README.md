Afin de convertir une solution en PDF:

1. Convertir la grille en LaTeX: `python3 exporter.py > hanoi.tex`
2. Compiler: par ex.             `pdflatex hanoi.tex`
3. Ouvrir `hanoi.pdf`

_La compilation requiert le package TikZ pour LaTeX._
